# CareFlow Hospital Management System

A modern hospital management platform designed to streamline patient management, doctor scheduling, operation theatre monitoring, blood bank management, and hospital analytics.

## Technologies Used
- React
- TypeScript
- Firebase
- Tailwind CSS
- Google Maps API

## Features
- Admin Dashboard
- Doctor Management
- Patient Records
- Operation Theatre Monitoring
- Blood Bank Management
- Responsive Design
